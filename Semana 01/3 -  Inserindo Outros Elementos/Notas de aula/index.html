Imagens (img)

Se percebe que as imagens são do tipo in-line block.

Links e Ancoras (a)

Podemos ver os 2 tipos de links:

  • Externo
  • Interno
  • Ancora

Temos também o atributo target que define o comportamento do navegador ao clicarmos no link:

  • _self: Abre o link na mesma aba, ou seja, o site será sobreposto pela página contida no link;
  • _black: Abre o link em uma nova guia (aba), ou seja, o site continuará aberto porém o levará para a nova aba aberta para que o usuário possa acessa-la.

Listas (ul/ol)

Temos dois tipos de listas:

  • Não ordenadas: ol
  • Ordenadas: ul
  • Descritivas: COMPLETAR

Listas Não Ordenadas UL

As marcações ul e /ul são utilizadas para gerar listas não ordenadas (Unordered lists). Estas tags devem envolver todo o conjunto de itens da lista, e cada item deve ser precedido da marcação li (Line item).

Listas não ordenadas possuem o parâmetro TYPE que podem ser dos seguintes tipos:

  • Square
  • Circle
  • Disc
  • Padrão

Vale mencionar que o parâmetro type pode ser usado tanto em ul como em li

Lista Ordenadas OL

As marcações utilizadas para gerar listas ordenadas (Ordered lists ) são ol e /ol. Estas tags devem envolver todo o conjunto de itens da lista, e cada item deve ser precedido da marcação li.

Assim como as Listas Não Ordenadas as Listas Ordenadas também possui o atributo type, no entanto, ela também possui o elemento start que define o numero que iniciara a lista.

  1. Padrão (1)
  2. Alfabética Maiúscula (A)
  3. Alfabética Minúscula (a)
  4. Alfabética Romana Maiúscula (I)
  5. Alfabética Romana Minúscula (i)

Lista de Definição

Uma lista de definições consiste em alternar um termo e uma definição. As tags dl e /dl são utilizadas para gerar listas de definições.

Estes comandos devem envolver todo o conjunto de termos e definições da lista. Os termos devem ser precedidos da marcação dt e as definições da marcação dd.
